# LUMEN ACADEMY — ART DIRECTION

## Direction
Experimental typography / playful digital education.

## Core metaphor
The website behaves like a learning compiler. Raw curiosity enters the system and becomes visible proof: projects, critiques, demos and career assets.

## Visual system
- Acid lime background: `#F3FF57`
- Ink: `#111111`
- Signal pink: `#FF4FBC`
- Electric blue: `#5B7CFF`
- Orange: `#FF6B35`
- Oversized condensed typography
- Editorial grids and deliberately hard borders
- Terminal and system-state details

## Motion language
- Letters appear as movable interface objects
- Code window reacts to pointer position
- Track data recompiles when the user changes direction
- Learning loop updates during scroll
- Projects travel horizontally on desktop
- Career evidence grows with a draggable slider

## Mobile behavior
Desktop horizontal scenes become vertical cards. Hover interactions always have tap equivalents. Cursor effects are disabled on touch devices.
